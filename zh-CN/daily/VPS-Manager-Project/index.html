VPS Manager 建项 🚩

发表于 2020-04-17 阅读次数： Valine：

💭 项目介绍/Intro

基于 Electron 的 自动化运维GUI运维的桌面工具。

☁️ 云平台维度

主要依靠云服务商提供的 API 接口来实现平台级的运维。

🖥 VPS维度

  • 简单的应用管理
    • 安装/卸载
    • 配置
    • …
  • 基础的系统安全
    • 防火墙
    • SSH禁用Root
    • …

与面板类型应用不同，不需要加装软件到VPS。也受限与于此，这类管控在监控方面主要比较欠缺，基本以来平台预装的流氓软件API基于 SSH/SCP 实现。

☑️ 功能计划/Road Map

🎉 V1.0 🎉

功能型

  • VPS 管理
    • 支持 Vultr API
    • 支持手动添加 VPS
  • APP 管理
    • 支持应用安装/卸载/更新/运行/停止（SSH/SCP）
    • 支持应用配置（SSH/SCP）
  • 支持 SCP
  • 支持 SSH
  • 一键配置 sudo 用户并禁用 ROOT 登录（SSH/SCP）
  • 生成 🔐

管理型

  • VPS/APP 标签分类管理
  • VPS/APP 标签自动分类
  • VPS/APP 手动排序（记忆性）
  • 用户信息独立管理（VPS 无关）
  • 钥匙对管理

交互型

  • 拖拽 VPS/APP 卡片实现切换分类/排序
